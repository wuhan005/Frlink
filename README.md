# Frlink
Frlink - Beautify WordPress Friends Links / WordPress 友链管理插件

## 功能

Frlink 是一款小巧的 WordPress 插件。其独立于 WordPress 原生的链接，使你可以管理自己站点的友情链接并能够以十分漂亮的形式呈现。
